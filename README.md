# Fact Check

A web-based application designed to verify and validate information, combining HTML frontend interfaces with Python backend logic for comprehensive fact-checking capabilities.

## 🎯 Overview

Fact Check is a tool that helps users verify the accuracy of claims and statements. With a balanced architecture of HTML (56.7%) and Python (43.3%), this project provides both an intuitive user interface and robust backend processing.

## ✨ Features

- **Web Interface**: Clean and intuitive HTML-based frontend for easy fact-checking
- **Python Backend**: Powerful backend logic for information verification and validation
- **User-Friendly Design**: Simple, accessible interface for all users
- **Efficient Processing**: Fast fact-checking algorithms and data processing

## 🛠️ Tech Stack

- **Frontend**: HTML (56.7%)
- **Backend**: Python (43.3%)
- **Architecture**: Web-based application

## 🚀 Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)
- A modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/dipanshu-c/fact-check.git
   cd fact-check
   ```

2. **Create a virtual environment** (recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**
   ```bash
   python app/main.py
   ```

5. **Access the application**
   Open your browser and navigate to `http://localhost:5000` (or the specified port)

## 📖 Usage

1. Enter a statement or claim you want to verify
2. Click the "Check Fact" button
3. View the verification results and supporting information
4. Review the confidence score and sources
